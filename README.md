\# 8-LED Sequential Chaser Using 8051 Microcontroller



\## Project Description



This project demonstrates the sequential blinking of 8 LEDs using an 8051 microcontroller. The LEDs glow one by one in a chaser pattern with a fixed delay. The program is written in Embedded C using Keil µVision and the circuit is simulated using Proteus 8 Professional.



\## Software Used



\* Keil µVision

\* Proteus 8 Professional



\## Components Used



\* 8051 Microcontroller

\* 8 LEDs

\* Resistors

\* Power Supply

\* Connecting Wires



\## Working



The 8051 microcontroller sends output signals through Port 1. Each LED is connected to one pin of Port 1. The LEDs turn ON one by one with a small delay, creating a sequential chaser effect.



\## Pin Connection



\* LED 1 → P1.0

\* LED 2 → P1.1

\* LED 3 → P1.2

\* LED 4 → P1.3

\* LED 5 → P1.4

\* LED 6 → P1.5

\* LED 7 → P1.6

\* LED 8 → P1.7



\## Applications



\* Basic embedded system learning

\* LED pattern display

\* Microcontroller output port practice

\* Proteus simulation practice



\## Output



The 8 LEDs blink continuously in a sequential pattern.



\## Conclusion



This project helps to understand the basic interfacing of LEDs with an 8051 microcontroller. It is useful for beginners to learn Embedded C programming, output port control, and circuit simulation using Keil and Proteus.
